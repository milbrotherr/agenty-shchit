# Правила работы с проектом «Агенты Щ.И.Т.»

Перед началом любой работы прочитай [КОНТЕКСТ ПРОЕКТА.md](КОНТЕКСТ%20ПРОЕКТА.md), основное досье и реестр материалов.

- Файлы в этой папке — постоянная память проекта. Сохраняй новые решения и материалы в проекте.
- Не дописывай лор от себя. Неустановленные сведения отмечай как `[НЕ ПОДТВЕРЖДЕНО]`; закрытые — как `[ЗАШИФРОВАНО]`.
- Поддерживай простой официальный стиль досье, если не дано другое указание.
- Не публикуй черновые визуальные материалы как утверждённые без явного запроса пользователя.
- При изменениях сайта сохраняй их в Git и публикуй в существующий публичный репозиторий только по запросу или в рамках явно запрошенного обновления сайта.
